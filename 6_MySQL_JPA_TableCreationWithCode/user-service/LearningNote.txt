A. Database operation i have done
--------------------
[in windows cmd prompt, in this sequence ]

* login:
mysql -u root -p;

* see existing databases
show databases;

* Create new db (jmslearn = ava microservice learn)
create database jmslearn;

* Check the entry of newly created db
show databases;

* Activate newly created db
use jmslearn;

* Create "users" table
CREATE TABLE users (
	id INT AUTO_INCREMENT PRIMARY KEY,
	first_name VARCHAR(255),
	last_name VARCHAR(255),
	email VARCHAR(255)
);

* see all the tables in activated db
show tables;

* check the schema of "users" table
desc users

* delete the table
drop table users;
